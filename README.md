# AG_News文本分類
***介紹**<br>
`AG News Dataset 擁有超過100萬篇的新聞文章`<br>

***目的**<br>
`使用 Transformer 對 AG News Dataset 進行分類任務`<br>
`將新聞分四大類 world, sports, business, and Science`<br>
